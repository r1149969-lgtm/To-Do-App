## To DO App

## Description 
the To Do App is a simple Python project that helps users manage their daily tasks.
Users can ad new tasks, view all tasks, and delete completed tasks. The project also uses file handling to save tasks, 


## Features
- Add a new task
- View all tasks
- Delete a task
- Exit the application

## Concept Used
- Lists
- Loops
- Conditional Statements(if-elif-else)
- User Input

## How to Run
Clone this repository.
open the project in vs code.
run the python file:
   '''bash
     python to-do.py
       '''
