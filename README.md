### Variables
 Definition: A variables in Python it act like a container it stores different types of data value.
 #### Syntax for variable Assignment
 x = 5  # Assigning  to integer value to the variable x
 y =  "Hello"  # Assigning string value to the variable y

 ### Variable Naming Rules
 .Variable names can contain letters(a-z,A-Z),numbers(0-9) or underscore(_)
 .Variables names must start with an letter or an underscore
 .Variables names are case-sensitive(Name and name are different)
